Getting AI working

Pick a provider in the AI tab and paste an API key. The model field auto-fills with a sensible default — change it from the dropdown if you like.

Where to get API keys

  • Anthropic (Claude): console.anthropic.com → API Keys
  • OpenAI (GPT): platform.openai.com → API keys
  • Google (Gemini): aistudio.google.com → Get API key
  • ElevenLabs (voice): elevenlabs.io → Profile → API key

Free / offline option (Ollama)

Install from ollama.com, then in a terminal run ollama pull llama3.2 (or llama3.2-vision for the Vision feature). Choose Ollama as the provider and hit Test. No key, fully local and private.

Where to get characters (VRM models)

  • VRoid Hub — hub.vroid.com (many free, check licenses)
  • VRoid Studio — make your own free custom model
  • Booth.pm — search 無料 (free) VRM models

Add one by dropping a model file (.vrm, .glb, .gltf, .fbx, .obj) or a .zip containing one onto the box in the Appearance tab. VRM models get full expression/eye-tracking support; other formats render and animate but without VRM-specific features.

Moving the character

Drag the character (or the ⠿ handle) to move it. In Appearance, turn on Wander to let it roam, Follow the cursor to chase your mouse, or click Come here to call it over once.

Handy

Global hotkey Alt+Shift+L shows/hides it anywhere. The 👁 button (with Vision on) lets it look at your screen.
